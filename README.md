## Installation

### Clone repo to destinaton.

```bash
git clone https://github.com/WebDevMullins/portfolio-react.git
```

### Install dependencies using:

```bash
npm install
```

## Usage

From the root directory, run the following to start the application:

```bash
npm run dev
```
